# BizPulse ERP - Fresh Deploy

## Quick Deploy to Render.com

1. Upload this folder to GitHub
2. Connect to Render.com
3. Deploy settings:
   - Build: `pip install -r requirements.txt`
   - Start: `gunicorn app:app --bind 0.0.0.0:$PORT`
